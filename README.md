# 🎯 AHP Decision Studio

Une application SaaS premium pour la prise de décision structurée basée sur la méthode **AHP (Analytical Hierarchy Process)**.

[![Next.js](https://img.shields.io/badge/Next.js-16-black?style=for-the-badge&logo=next.js)](https://nextjs.org/)
[![React](https://img.shields.io/badge/React-19-61DAFB?style=for-the-badge&logo=react)](https://react.dev/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind--CSS-4.0-38B2AC?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com/)

## ✨ Fonctionnalités
- 🚀 **Onboarding Fluide** : Expérience utilisateur immersive dès le premier lancement.
- 📊 **Tableau de Bord Moderne** : Navigation libre entre critères, alternatives et analyses.
- ⚖️ **Matrice de Saaty** : Interface intuitive pour les comparaisons par paires avec validation de cohérence (CR).
- 🏆 **Classement Optimal** : Visualisation claire de la meilleure option via des graphiques et scores pondérés.
- 🌓 **Mode Sombre** : Support complet des thèmes clair et sombre.

## 🛠️ Installation

```bash
# Cloner le projet
git clone https://github.com/JordanFotso/classification.git

# Installer les dépendances
npm install

# Lancer le serveur
npm run dev
```
