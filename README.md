# AHP Decision Studio

Une application SaaS pour la prise de décision structurée basée sur la méthode **AHP (Analytical Hierarchy Process)**.

## Lien de Déploiement
**Accéder à l'application en ligne :** [ahp-uy-1-ine-sigl-4.vercel.app](https://ahp-uy-1-ine-sigl-4.vercel.app/)

## Fonctionnalités
- **Onboarding Fluide** : Expérience utilisateur immersive dès le premier lancement.
- **Tableau de Bord Moderne** : Navigation libre entre critères, alternatives et analyses.
- **Matrice de Saaty** : Interface intuitive pour les comparaisons par paires avec validation de cohérence (CR).
- **Classement Optimal** : Visualisation claire de la meilleure option via des graphiques et scores pondérés.
- **Mode Sombre** : Support complet des thèmes clair et sombre.

## Installation

```bash
# Cloner le projet
git clone https://github.com/JordanFotso/AHP-UY1-INE-SIGL4.git

# Installer les dépendances
npm install

# Lancer le serveur
npm run dev
```
